Статус: ✅Сдано

---
## Постановка задачи:

Дана функция: $$y = \ln\left(\frac{1}{2 + 2x + x^2}\right)$$
Разложить функцию в степенной ряд и вычислить её значение двумя способами:

1. По формуле (функция y)

2. Через разложение в ряд (сумма S)

Сравнить результаты, убедиться в их совпадении.

**Дано:**

- x — значение аргумента

- n = 40 — количество членов ряда
  
**Найти:**

- S — сумма ряда

- Y — значение по формуле

## Исходный код:

```cpp
#include <iostream>
#include <clocale>
using namespace std;

double func_n(double n, double x, int i, double a1) 
{
	if (i == n)
	{
		return a1;
	}
	double an = a1 * ((-(1 + x) * (1 + x)) * (i / (i + 1.0)));
	return a1 + func_n(n, x, i + 1, an);
}
double func_y(double x) 
{
	return log(1.0 / (2.0 + 2.0 * x + x * x));
}
int main() 
{
	setlocale(LC_ALL, "ru");
	int n = 40;
	double x;
	cout << "Введите x в диапазоне [-2;-0.1]: ";
	cin >> x;

	double a1 = -(1 + x) * (1 + x);
	cout << "\nX = " << x << '\t'
		<< "S = " << func_n(n, x, 1, a1) << '\t'
		<< "Y = " << func_y(x) << endl;
	return 0;
}
```
